## Project Structure
- Modern React dashboard with sidebar navigation
- Responsive design using Tailwind CSS
- Vite for fast development and building
- Component-based architecture
- Power BI integration ready

## Components
- `Sidebar.jsx` - Navigation sidebar with menu items
- `Header.jsx` - Top header with user info and notifications
- `StatsGrid.jsx` - Dashboard statistics cards
- `PowerBIEmbed.jsx` - Dedicated Power BI dashboard embedding area
- `RecentActivity.jsx` - Activity feed component

## Getting Started
1. Install dependencies: `npm install`
2. Start development server: `npm run dev`
3. Build for production: `npm run build`

## Power BI Integration
The dashboard includes a dedicated `PowerBIEmbed` component ready for Power BI dashboard integration. To set up:
1. Configure your Power BI workspace
2. Generate embed tokens
3. Update the component with your report URL and credentials
4. Test the embedding functionality

## Development
- The development server is configured to run on `npm run dev`
- Tailwind CSS is fully configured with custom color palette
- All components are responsive and professional-looking

## Technical Details
This template uses:
- [Vite](https://vite.dev) for fast development and building
- [React](https://react.dev) for the UI framework
- [Tailwind CSS](https://tailwindcss.com) for styling
- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) for Fast Refresh
